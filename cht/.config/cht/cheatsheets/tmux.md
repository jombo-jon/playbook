# TMUX

## General
C-a + c     : Create a new window
C-a + x     : kill the current pane
C-a + &     : kill the session
C-a + ?     : show list of keybindings
C-a + [     : Enter copy mode
C-a + ]     : Paste the most recent paste buffer

## Split 
C-a + s     : split horizontal
C-a + d     : split vertical
